RenderContext Klassenreferenz

Ein RenderContext hält alle Informationen, die für einen Renderdurchlauf über die Entities hinaus wichtig sind. Das ist immer mindestens eine Kamera. Darüber hinaus können es verschiedene Rendertargets oder die Konfiguration der Buffer, in die gerendert werden soll, sein.

#include <rendercontext.hpp>
