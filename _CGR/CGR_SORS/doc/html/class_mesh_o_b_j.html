MeshOBJ Klassenreferenz

Sehr einfache Implementierung einer Mesh-Klasse, die aus .obj-Dateien (z.B. aus Blender) geladen wird. Zum Laden wird tinyobjloader (siehe thirdparty) verwendet. Vorsicht: Es werden derzeit nur Meshes geladen, die aus einem Teil bestehen. Weitere Einschränkunen sind, dass nur die diffuse Textur und keine Materialien geladen werden.
