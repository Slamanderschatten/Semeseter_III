Trackball Klassenreferenz

Diese Klasse realisiert einen einfachen Trackball und kann als Controller zu Entities hinzugefügt werden. Mit gedrückter linker Maustaste wird dann das jeweilige Entity entsprechend rotiert.

#include <trackball.hpp>
